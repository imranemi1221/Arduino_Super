# Arduino_Super is Soluation for the Building lights control
##It's an Arduino based Bluetooth control Android project.

##Using Equipment: 
1. Arduino uno
2. Android mobile 
3. Bluetooth receiver HC-06
4. Connecting wire
5. Different color LED
